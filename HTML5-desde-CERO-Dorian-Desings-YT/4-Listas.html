Tipos de listas en HTML

Lista desordenada: Ejemplo - Lista de compras

  • Tomate
  • Jitomate
  • Huevo
  • Jamón
  • Salchicha
  • Crema
  • Tocino
  • Miel

Otro ejemplo: uso de una lista desordenada para crear un menú de navegación.

  • Inicio
  • Sobre mí
  • Servicios
  • Contacto

Lista ordenada: Ejemplo - Pasos para preparar un sándwich

  1. Tostar el pan
  2. Untar mayonesa
  3. Agregar jamón y queso
  4. Colocar lechuga y tomate
  5. Cerrar el sándwich
  6. Cortar por la mitad y servir

Lista de definición: Ejemplo - Glosario de términos

HTML
Lenguaje de marcado utilizado para estructurar páginas web.
CSS
Lenguaje de hojas de estilo utilizado para dar formato y diseño al contenido HTML.
JavaScript
Lenguaje de programación que permite agregar interactividad a las páginas web.
